# amazon-clone
A fully functional Amazon Clone built with the MERN stack (or React/Firebase). Features include user authentication, a real-time product search, shopping cart functionality, and integrated payment processing with Stripe.

# Amazon Clone - Full Stack

A modern e-commerce web application built to demonstrate full-stack development skills.

## 🚀 Tech Stack
* **Frontend:** React.js (Vite), CSS/Tailwind
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Authentication:** Firebase Auth / JWT
* **Payments:** Stripe API

## 🛠️ Installation & Setup
1. Clone the repository
2. Run `npm install` in the root and client folders
3. Create a `.env` file for your API keys
4. Run `npm run dev` to start the development server
